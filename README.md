🧮 Calculator Web App

A simple and responsive Calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations with a clean and modern user interface.

---

🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Percentage (%)
- 🧹 Clear (C) button
- ⌫ Delete (DEL) button
- ⚡ Instant result calculation
- 📱 Responsive design

---

📁 Project Structure

calculator/
│── index.html
│── css/
│   └── style.css
│── js/
│   └── script.js

---

🛠️ Technologies Used

- HTML5
- CSS3 (Grid Layout)
- JavaScript (DOM Manipulation)

---

⚙️ How It Works

- User clicks buttons to input numbers and operators
- JavaScript updates the display dynamically
- "eval()" function is used to calculate the result
- Error handling prevents invalid calculations

---

▶️ How to Run

1. Download or clone the project:

git clone https://github.com/your-username/calculator.git

2. Open the folder

3. Run the project:

- Open "index.html" in your browser

---

⚠️ Important Note

This project uses:

eval()

for calculations.

👉 This is fine for basic projects, but not recommended for production apps due to security risks.
